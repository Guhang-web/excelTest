# 📊 Excel Level Test (엑셀 레벨 테스트)

상담 전 고객의 엑셀 활용 능력을 빠르고 정확하게 진단하기 위한 웹 어플리케이션입니다. 
7개의 문항을 통해 기초부터 실무 활용 능력까지 측정합니다.

## 🚀 주요 기능
- **단일 페이지 전환 (SPA):** 태블릿에서도 끊김 없는 사용자 경험 제공
- **찍기 방지:** 모든 문제에 '6번 모름' 보기를 추가하여 정확한 진단 유도
- **오답 노트:** 테스트 종료 후 틀린 문제에 대한 정답과 상세 해설 제공
- **반응형 디자인:** 모바일 및 태블릿 터치에 최적화된 UI

---

## ✅ 테스트 문제 정답 및 해설 (관리자용)

| 번호 | 영역 | 정답 | 설명 요약 |
|:---:|:---:|:---|:---|
| **Q1** | 기초 | **4. SUM** | 범지의 합계를 구하는 가장 기본 함수 |
| **Q2** | 참조 | **3. $** | 수식 복사 시 셀 주소를 고정하는 절대 참조 기호 |
| **Q3** | 논리 | **3. IF** | 조건에 따라 참/거짓 결과값을 반환하는 함수 |
| **Q4** | 실무 | **5. VLOOKUP** | 특정 기준값으로 다른 표에서 데이터를 찾아오는 함수 |
| **Q5** | 가공 | **1. TRIM** | 텍스트 양 끝 및 불필요한 공백을 제거하는 함수 |
| **Q6** | 분석 | **3. 피벗 테이블** | 대량의 데이터를 요약하고 분석하는 강력한 도구 |
| **Q7** | 단축키 | **2. Ctrl + Shift + L** | 필터(Filter) 기능을 빠르게 적용/해제하는 단축키 |

## 🛠 기술 스택
- **Language:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Hosting:** GitHub Pages
- **Design:** Excel Branding (Green Theme)

## 📁 파일 구조
- `index.html`: 전체적인 구조 및 UI 섹션
- `style.css`: 엑셀 테마 디자인 및 레이아웃
- `script.js`: 퀴즈 로직, 채점 및 오답 노트 생성
- `main.jpg`: 상단 메인 배너 이미지(img폴더)
